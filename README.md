### 🌍 Global Layoffs Analysis (MySQL)
**[View Repository](https://github.com/louischidebe/global-layoffs-analysis-sql)**

An end-to-end SQL project exploring global tech layoffs between 2020 and 2026.

**Highlights**
- Cleaned the dataset by removing duplicates, handling missing values, standardizing company names and locations, and formatting dates.
- Used CTEs, window functions, and aggregate queries to identify trends across industries, countries, and companies.

**Key Findings**
- The United States recorded the highest number of layoffs.
- 2023 had the highest annual layoff total.
- Seed-stage companies experienced the highest rate of complete shutdowns.

**Tools:** MySQL

### 🏥 Hospital ER Performance Analysis (Power BI)
**[View Repository](https://github.com/louischidebe/hospital-er-performance-analysis)**

An interactive Power BI dashboard built from 19 months of emergency room data covering more than 9,000 patient records.

**Highlights**
- Cleaned and transformed data with Power Query.
- Built a relational data model with a dedicated Date table.
- Created DAX measures and KPIs to monitor wait times, patient satisfaction, referrals, and demographic trends.

**Key Findings**
- Average wait time was 35.3 minutes, above the 30-minute target.
- Only 40% of patients were seen within the target wait time.
- Patient traffic was consistently highest between midnight and 2 a.m.

**Tools:** Power BI, Power Query, DAX
